
# Requirement
- maven
- java 17+
- postgres 
# Tehnology
- Spring Boot 3
- Spring Security 6
- Lombok 
- Jwt Token
# How to install liblary
``mvn clean install``
# How to run
```mvn -pl omnierp-app spring-boot:run```
# Make jar
```mvn clean install package```<br>
```java -jar omnierp-app/target/omnierp-app-0.0.1-SNAPSHOT.jar```
