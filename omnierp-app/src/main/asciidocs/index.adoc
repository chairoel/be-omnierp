


= Module Auth API

=== REGISTER API

|===
|POST |/api/v1/user/regis

| Request Body
| {
"firstname": "Kuda";
"lastname": "Nil";
"email": "kuda.nil@gmail.com";
"roles": ["USER"];
"password": "password";
}

| Response Body
| {
"email": "test.test@gmail.com",
"password": "password"
}
|===
